# Split Stockholm format into FASTA

```
$ cargo run -- -h
Split Stockholm format into FASTA

Usage: stksplit [OPTIONS] [FILENAME]

Arguments:
  [FILENAME]  Input file [default: -]

Options:
  -o, --outdir <OUTDIR>  Output directory [default: out]
  -h, --help             Print help
  -V, --version          Print version
```

Cf `make run` for an example.
